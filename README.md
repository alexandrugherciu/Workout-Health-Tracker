# Workout-Health-Tracker
Full-stack fitness and health tracking platform with workout management, authentication, and PostgreSQL integration.

## Features

- User authentication and authorization
- Workout tracking and management
- Health-related statistics
- REST API integration
- Responsive user interface

## Technologies Used

### Frontend
- React
- TypeScript
- HTML/CSS

### Backend
- ASP.NET Core
- C#
- REST API

### Database
- PostgreSQL

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
dotnet restore
dotnet run
```
